# Treezer Hellen's Portfolio

![Website Preview](images/hero-bg.jpg)

A vibrant portfolio showcasing Treezer Hellen's web development expertise and passion for Luo cuisine.

## Features

- Interactive profile with traditional attire toggle
- Animated skill bars
- Authentic Kenyan recipes with Luo influence
- Responsive design for all devices
- Cultural elements throughout design

## Pages

1. *Home* - Introduction and featured recipes
2. *About* - Interactive profile with skills
3. *Ugali & Fish* - Traditional Luo recipe
4. *Chapati & Beans* - Coastal-inspired recipe

## Setup

1. Clone repository
2. Add these images to /images:
   - hero-bg.jpg
   - profile.jpg
   - traditional.jpg
   - ugali-fish.jpg
   - chapati.jpg
3. Open index.html in browser
